# 복지이음 (WelfareEum)

**LLM + RAG 기반 개인 맞춤형 음성 복지 서비스**

복지이음은 사용자의 연령, 가구형태, 소득, 지역 등 상황 정보를 분석하여  
가장 적합한 복지 제도를 추천하는 대화형 AI 서비스입니다.

---

## 프로젝트 개요

- **목표**  
  디지털 접근성이 낮은 이용자도 음성을 통해 쉽고 자연스럽게 복지 제도를 탐색할 수 있는 서비스 구축

- **핵심 기술**  
  LLM 기반 자연어 이해, RAG 기반 의미 검색, STT/TTS 음성 인터페이스

- **주요 기능**
  - 음성 및 텍스트 입력 기반 복지 질의응답
  - 개인 맞춤형 복지 제도 추천
  - 게스트 모드 진입 및 설문형 탐색 지원
